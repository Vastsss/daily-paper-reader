## 摘要
本文证明了当m>2n时，一个n维混合系统可以嵌入到一个配备有连续向量场的m维欧几里得空间的嵌入像中。这一结果表明，本质上不连续的混合系统通常允许一个连续的、适用于可微优化的外部表示。基于这一存在性定理，我们展示了一个在潜空间和状态空间都带有一致性损失的潜神经ODE能够准确恢复混合系统的流。大量实验表明，所提出的方法在仅从时间序列数据学习具有不同几何结构的混合系统方面优于现有方法。

## Abstract
This work proves that an $n$-dimensional hybrid system can be embedded into an $m$-dimensional Euclidean space equipped with a continuous vector field on its embedded image whenever $m>2n$. This result suggests that an intrinsically discontinuous hybrid system generically admits a continuous extrinsic representation that is well-posed for differentiable optimization. Building on this existence theorem, we show that a latent Neural ODE with consistency loss in both the latent and state space can accurately recover the flow of hybrid systems. Extensive experiments suggest the proposed method outperforms the existing method in learning hybrid systems with varying geometries from only time series data.

### 2. 论文提出的方法论

-   **核心思想**：通过一个精心设计的嵌入函数 $f$，将混合系统的状态空间 $M$ 映射到一个更高维的潜空间 $\mathbb{R}^m$。这个嵌入要满足两个关键条件，使得在潜空间中，原本不连续的混合动态变为连续的：
    1.  **值兼容性 (C-1)**：在守卫面（guard）上，碰撞前状态 $x$ 和碰撞后状态 $r(x)$ 被映射到潜空间的同一点，即 $f(x) = f(r(x))$。这“粘合”了状态空间的不连续点。
    2.  **导数兼容性 (C-2)**：在守卫面上，碰撞前后的向量场导数在潜空间中保持一致，即 $\frac{d}{dt}f(x) = \frac{d}{dt}f(r(x))$。这确保了潜空间中的向量场穿越守卫面时仍然连续。

-   **关键技术细节**：
    -   **理论基础**：论文利用微分拓扑中的**横截性定理**和**Whitney嵌入定理**。证明的核心是，当潜空间维数 $m > 2n$（$n$是原始状态空间维数）时，满足上述两个条件的嵌入函数 $f$ 是普遍存在的（generic），即不是特例而是一大类函数的共同属性。
    -   **构造性证明**：
        1.  首先在守卫面 $S$ 和其像 $r(S)$ 的邻域（collar neighborhood）上构造 $f$。

        3.  使用横截性理论证明，对于“大多数” $g_S$ 的选择，$f$ 在该邻域内是浸入且单射的。
        4.  最后，通过一个相对扰动（relative perturbation）技巧，将该局部性质扩展到整个状态空间 $M$，保证 $f$ 在整个流形上是嵌入。

        -   基于上述理论，提出一个**潜神经ODE（Latent Neural ODE）**框架。
        -   **编码器 $f_\theta$**：将状态 $x$ 映射到潜空间 $z$。
        -   **潜向量场 $V_\theta$**：在潜空间中由神经网络定义的连续向量场。
        -   **解码器 $f_\theta^{-1}$**：将潜变量 $z$ 映射回原始状态 $x$。

            1.  **状态一致性损失 $L_x$**：解码后的轨迹与真实轨迹的MSE。
            2.  **潜空间一致性损失 $L_z$**：潜空间预测轨迹与编码后的真实轨迹的MSE。
            3.  **粘合损失 $L_g$**：显式地强制碰撞前后状态在潜空间中被映射到同一点，对应条件 (C-1)。
            4.  **速度兼容性损失 $L_v$**：近似地强制碰撞前后潜变量的变化率一致，对应条件 (C-2)。
            5.  **协方差损失 $L_c$**：防止潜空间坍缩，鼓励其分布有良好的结构。
