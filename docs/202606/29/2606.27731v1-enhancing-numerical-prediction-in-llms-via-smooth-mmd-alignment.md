## 摘要
尽管大语言模型（LLM）具有强大的通用能力，但在输出必须精确数值时往往不可靠。一个关键原因是训练目标：标准交叉熵将数值标记视为无结构类别，忽略其数值的度量结构。我们通过平滑最大均值差异（SMMD）来解决这一不匹配问题，该方法在经典MMD基础上引入数值标记上的值距离核和图基平滑。通过在该数值子词汇表上定义核，SMMD通过核匹配将预测的数值分布与目标对齐，并在诱导核图上平滑预测-目标残差以鼓励局部一致性。我们在四个数值目标任务上评估SMMD：数学推理、算术计算、时钟时间识别和图表问答，使用多个开放权重的LLM和VLM骨干网络。SMMD在交叉熵和最近的数值目标损失上持续提高准确率；分析显示了MMD和平滑之间的互补效应，并强调了基于距离的核设计的重要性。代码可在https://github.com/Zuozhuo/smmd-loss获取。

## Abstract
Despite their strong general capabilities, large language models (LLMs) often remain unreliable when outputs must be numerically precise. A key reason is the training objective: standard cross-entropy treats numeric tokens as unstructured categories and ignores the metric structure of their values. We address this mismatch with Smooth Maximum Mean Discrepancy (SMMD), which builds on the classic MMD by incorporating value-distance kernels over numeric tokens and graph-based smoothness. With this kernel defined over a numeric sub-vocabulary, SMMD aligns the predicted numeric distribution to the target via kernel matching and smooths the prediction-target residual over the induced kernel graph to encourage local consistency. We evaluate SMMD on four numeric-target tasks: mathematical reasoning, arithmetic calculation, clock-time recognition, and chart question answering, across multiple open-weight LLM and VLM backbones. SMMD consistently improves accuracy over both cross-entropy and recent numeric-target losses; analyses show complementary effects between MMD and smoothness and underscore the importance of distance-based kernel design. Code is available at https://github.com/Zuozhuo/smmd-loss.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：大型语言模型在需要精确数值输出的任务中（如数学推理、时间识别、图表问答）常常不可靠。根本原因是标准交叉熵（Cross-Entropy, CE）训练目标将数值 token 视为无结构类别，忽略数值之间的度量关系（如距离、序关系）。例如，混淆“3”与“4”与混淆“3”与“7”受到同样的惩罚，模型无法感知数值临近性。
- **研究动机**：数值错误在科学、金融、决策等场景中会传播并导致严重后果，需要一种能反映数值度量结构且易于集成的训练损失。
- **整体含义**：通过对数值预测分布进行基于核的分布匹配和局部平滑，使模型输出更精确、更符合数值空间的结构。

## 2. 方法论：核心思想、技术细节、算法流程（文字说明）
- **核心思想**：提出 Smooth Maximum Mean Discrepancy (SMMD) 损失，构建数值子词汇上的距离诱导核，利用核 MMD 对齐预测分布与目标分布，并在核诱导图上施加平滑正则化以促进局部一致性。
- **关键技术细节**：
  - **数值子词汇构建**：预先从全词汇中筛选可解析为实数的 token，构成 `V_num`（如单数字 token 或多位整数 token），建立索引到值的映射。
  - **距离诱导核**：基于 token 值的绝对距离，使用 RBF（径向基函数）核或多种带宽的混合核，生成对称正定 Gram 矩阵 `K`，使得数值越近的 token 相似度越高。
  - **核 MMD 对齐**：定义残差 `r = p - q`（预测分布减 one-hot 目标），平方 MMD 损失为 `L_MMD = r^T K r`。当 `K` 正定时，零损失等价于 `p = q`。
  - **平滑正则化**：在核诱导图上定义 Dirichlet 能量 `L_smooth = r^T L r`，其中 `L = D - K` 为图拉普拉斯矩阵。该正则项惩罚残差在邻近数值上的突变，鼓励局部一致性。
  - **统一目标**：SMMD 损失为 `L_SMMD = r^T (K + αL) r`，整体训练损失为 `L = L_CE + λ L_SMMD`。α 自动设为 `1/(2\bar{d})`（`\bar{d}` 为平均度）。仅在目标 token 属于 `V_num` 时激活该损失。
- **算法流程**（文字描述）：
  1. 预计算数值子词汇 `V_num` 及其值映射。
  2. 预计算核矩阵 `K` 和拉普拉斯矩阵 `L`，以及 α。
  3. 训练时，对每个 minibatch：
     - 前向传播得到全词汇 logits。
     - 计算标准交叉熵 `L_CE`。
     - 对每个目标 token 在 `V_num` 内的位置，限制 logits 到 `V_num`，softmax 得到 `p`，构建 one-hot `q`，计算 `r`。
     - 累加 `r^T K r + α r^T L r`，平均后得 `L_SMMD`。
     - 总损失 `L_CE + λ L_SMMD`，反向传播更新参数。

## 6. 主要结论与发现
- SMMD 在四个数值任务上持续优于 CE 及所有对比的数值感知损失（GCE、NTL、NTIL），尤其在高精度任务（时钟时间识别）上提升显著（如 Ministral-3-3B 从 82.36% 到 97.56%）。

- 基于真实数值距离的核设计是关键，随机 PSD 或打乱映射均显著降低性能。
- 单核（σ=2）效果优于多核混合，说明校准核的局部性比增加核复杂度更重要。
- SMMD 引入的训练开销很小（相对于 CE 增加约 2-3% 步时间），内存增加类似其他数值损失。
- SMMD 不损害非数值能力（SAMSum、MMLU 保持接近）。
