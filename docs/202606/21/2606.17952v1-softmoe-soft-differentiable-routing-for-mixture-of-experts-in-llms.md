## 摘要
稀疏混合专家架构通过 top-$k$ 路由仅激活一小部分专家，从而在固定推理预算下扩展大语言模型参数。虽然这保持了因果性并适用于自回归语言模型，但离散的 top-$k$ 算子不可微，导致每个输入激活固定数量的专家，计算效率低下。我们提出 SoftMoE，用截断的软 top-k LapSum 松弛替代离散路由，实现基于梯度的专家路由优化。我们进一步参数化每层平均激活专家数并施加全局预算约束，使模型学习如何跨层分配专家容量。SoftMoE 与自回归建模完全兼容，在语言建模和下游任务上达到与稀疏 MoE 相当或更优的性能，同时激活显著更少的专家。值得注意的是，学习到的分配高度不均匀，后面层激活更多专家。源代码已公开$^\dagger$。

## Abstract
Sparse Mixture-of-Experts (MoE) architectures enable scaling LLM parameters under a fixed inference budget by activating only a small subset of experts via top-$k$ routing. While this preserves causality and suits autoregressive language models, the discrete top-$k$ operator is not differentiable, forcing a fixed number of active experts per input and resulting in inefficient use of computation. We propose SoftMoE, which replaces discrete routing with a truncated soft top-$k$ LapSum relaxation, allowing gradient-based optimization of expert routing. We further parameterize the mean number of active experts per layer and impose a global budget constraint, enabling the model to learn how to allocate expert capacity across layers. SoftMoE remains fully compatible with autoregressive modeling and achieves performance comparable to or better than sparse MoE on language modeling and downstream tasks, while activating significantly fewer experts. Notably, the learned allocation is highly non-uniform, with later layers activating more experts. The source code is publicly available$^\dagger$.

### 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：稀疏混合专家（MoE）架构通过离散的 top-$k$ 路由激活部分专家，以在固定推理预算下扩展模型参数。但离散 top-$k$ 算子不可微，迫使每层、每 token 激活固定数量的专家，导致计算资源利用不高效，且无法自适应分配专家容量。
- **研究动机**：现有方法（如专家选择路由、线性分配、最优传输等）虽试图解决离散路由的局限，但往往破坏自回归因果性、解耦训练与推理路由，或造成推理负载不均。作者希望设计一种可微分、保持因果性、且能自适应分配专家预算的路由机制。
- **整体含义**：SoftMoE 通过可微分的软 top-$k$ 松弛，让模型学会按层、按 token 动态分配专家，在保持性能的同时显著降低激活专家数，提升计算效率。

### 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：用截断的软 top-$k$ LapSum 松弛替代离散 top-$k 路由，使得路由选择可微，从而允许梯度优化专家分配；同时参数化每层平均激活专家数，并施加全局预算约束，让模型自动学习跨层预算分配。
- **关键技术细节**：
  - **LapSum 松弛**：利用 Laplace 累积分布函数之和构建函数 `LapSum(x) = Σ FLap(r_i - x)`，求解 `LapSum(x)=k` 得到可微阈值 $\tilde{x}$，然后软选择权重为 $\tilde{p}_i = F_{\text{Lap}}(r_i - \tilde{x})$。该操作复杂度 O(n)，无需排序，且梯度可传回路由分数 $r$ 和选择参数 $k$。
  - **截断操作**：对软权重 $\tilde{p}$ 施加阈值 $\tau$，仅保留大于 $\tau$ 的专家（梯度通过，小于 $\tau$ 的梯度为 0），实现稀疏性且激活专家数依输入变化。
  - **学习跨层分配**：引入每层平均激活专家数 $k_l$，通过重新参数化：$\pi = \text{softmax}(\eta)$，$k_l = \pi_l \cdot (K-L) + 1$，其中 $K$ 为全局总预算，$L$ 为层数。约束 $\sum_l k_l = K$ 且 $k_l \ge 1$。$\eta$ 可端到端训练。
  - **兼容自回归**：每个 token 独立路由，不涉及跨 token 混合，保持因果性。

### 6. 论文的主要结论与发现
- **主要结论**：
  - SoftMoE 在语言建模损失和下游任务上达到与稀疏 MoE 相当或更优的性能，同时激活的专家数显著减少（例如在 OWT 上，SoftMoE* 比 top-2 MoE 损失更低但激活专家数减少 17~24%）。
  - 学习到的专家分配高度不均匀：最后几层获得更多预算（约 50% 预算分配给 top-3 层），早期和中间层分配较少。

- **发现**：模型倾向于将更多计算资源分配给深层（语义层），与先前研究（BERT 深层编码语义信息）相符，为 MoE 深度分配提供新见解。
