## 摘要
时间序列基础模型在给定非空历史窗口时表现出强大的迁移性能。然而，真正的冷启动场景（即新项目没有先前观测数据）违背了这一假设。我们提出了RAID（检索增强迭代扩散）框架，该框架用元数据驱动的语义检索和图条件扩散取代了基于历史的相关性学习。RAID使用冻结的多语言嵌入模型将文本元数据映射到共享语义空间，并构建一个可自然扩展到未见项目的归纳检索图。它首先通过聚合语义相关邻居的信息形成基础预测，然后使用门控扩散模块细化该预测以建模残差不确定性。在严格真正冷启动协议下，RAID在预测准确性和预测区间覆盖率方面均优于强大的基础模型和竞争基线，同时通过非自回归解码将推理延迟降低一个数量级。共享语义空间还实现了零样本跨语言迁移，使得在英文描述上训练的模型能够泛化到用其他语言描述的项目，而无需直接监督。

## Abstract
Time-series foundation models show strong transfer performance when given a non-empty history window. However, true cold-start scenarios, where a new item has no prior observations, violate this assumption. We propose RAID (Retrieval-Augmented Iterative Diffusion) a framework, which replaces history-based correlation learning with metadata-driven semantic retrieval and graph-conditioned diffusion. RAID maps textual metadata into a shared semantic space using a frozen multilingual embedding model and constructs an inductive retrieval graph that extends naturally to unseen items. It first forms a base forecast by aggregating information from semantically related neighbors, then refines this forecast with a gated diffusion module to model residual uncertainty. Under a strict true cold-start protocol, RAID outperforms strong foundation models and competitive baselines on both forecasting accuracy and prediction interval coverage, while reducing inference latency by an order of magnitude through non-autoregressive decoding. The shared semantic space also enables zero-shot cross-lingual transfer, allowing a model trained on English descriptions to generalize to items described in other languages without direct supervision.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：现有时间序列基础模型（如Chronos、TimesFM）在零样本预测中依赖**非空历史窗口**（L > 0），无法处理**真正冷启动（True Cold-Start）**场景——即新项目没有任何历史观测数据（L = 0）。此时模型退化为全局均值或随机噪声，无法利用可用的文本元数据（如产品描述、规格等）。
- **整体含义**：冷启动预测在电商、云计算资源分配等领域至关重要，而现有方法要么依赖历史要么缺乏有效元数据融合。论文提出了**RAID**（Retrieval-Augmented Iterative Diffusion）框架，通过语义检索和图扩散，使得仅凭元数据就能生成高质量的预测，并且支持零样本跨语言迁移。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- **语义同质性（Semantic Homophily）**：相似元数据的物品通常具有相似的需求动态，因此可以通过语义相似的邻居传递需求信号。
- **检索与生成的解耦**：用**检索**构建基础预测，用**扩散**建模残差不确定性，从而支持非自回归的高效推理。

### 关键技术细节
1. **语义嵌入空间**：使用冻结的多语言嵌入模型**BGE-M3**将文本元数据映射到单位超球面，得到向量 \( e_i \)；再通过Tanh激活的瓶颈层得到低维条件特征 \( e'_i \)。
2. **归纳图拓扑**：基于Top-\(k\)稀疏softmax核构建图，权重 \( A_{ij} = \frac{\exp(\langle e_i, e_j \rangle / \tau)}{\sum_{l \in N_k(i)} \exp(\langle e_i, e_l \rangle / \tau)} \)。该图是归纳式的，新节点只需一次前向即可加入。
3. **检索增强基础预测（Shape-Scale分解）**：
   - 对每个邻居轨迹 \( y_j \) 进行归一化得到形状 \( z_j \)（z-score后截断）。
   - 用注意力权重聚合得到 \( z_{\text{agg}} \)。
   - 用MLP从瓶颈特征预测目标本身的尺度 \( \hat{\sigma}_i, \hat{\mu}_i \)。
   - 基础预测：\( \mu_{\text{base}}(t) = \text{Softplus}(\hat{\sigma}_i) \cdot z_{\text{agg}}(t) + \hat{\mu}_i \)。
4. **门控扩散Transformer（DiT）**：
   - 对残差 \( r = y - \mu_{\text{base}} \) 进行DDPM扩散。
   - 采用DiT骨干，用**自适应层归一化（AdaLN）**注入图上下文 \( c \) 和扩散步嵌入。
   - **门控融合**：最终预测 \( \hat{y} = \mu_{\text{base}} + \sigma(\text{MLP}_\psi(c)) \cdot \hat{r}_0 \)，门控初始偏向于基础预测（sigmoid(-3.0) ≈ 0.047）。
- **训练与推理流程**：见算法1和算法2。训练时模拟冷启动（mask历史），推理时只需 \( S=10 \) 步（非自回归），远少于传统扩散的50-100步。
