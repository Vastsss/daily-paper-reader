## 摘要
我们提出了分布感知共形预测（DCP），这是一个统一框架，将蒙特卡洛丢弃法、深度集成和分位数回归等概率预测器与分数无关的共形校准相结合，以生成有效且高效的预测区间。通过利用数值反演方法构建区间边界，DCP可以适应分布生成预测器和非一致性分数的任意组合。在合成和真实时间序列数据上的基准分析表明，DCP能够在不同不确定性条件下自适应地校准预测区间。关键的是，DCP的模块化设计便于对不同预测器-分数组合进行即插即用实验，并通过新引入的修正Winkler分数定量支持，该分数通过明确惩罚欠覆盖来平衡有效性和效率。虽然DCP推广并扩展了现有方法（如共形分位数回归和共形蒙特卡洛），但其模块化设计允许进一步扩展，为动态环境和高风险应用中的不确定性量化奠定了基础。

## Abstract
We present Distribution-aware Conformal Prediction (DCP), a unified framework integrating probabilistic predictors like Monte Carlo dropout, deep ensembles, and quantile regression with score-agnostic conformal calibration to produce valid and efficient prediction intervals. Leveraging a numerical inversion approach to construct interval bounds, DCP accommodates arbitrary combinations of distribution generating predictors and nonconformity scores. Benchmark analysis on synthetic and real-world time series data demonstrate DCP's ability to adaptively calibrate prediction intervals under varying uncertainty regimes. Crucially, DCP's modular design facilitates plug-and-play experimentation with different predictor-score pairings, quantitatively supported by a newly introduced modified Winkler score that balances validity and efficiency by explicitly penalizing undercoverage. While DCP generalizes and extends existing approaches like Conformalized Quantile Regression and Conformalized Monte Carlo, its modular design allows further extensions, setting a foundation for advancing uncertainty quantification in dynamic environments and high-risk applications.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 核心思想
DCP将概率预测器（DGP）与分数无关的共形校准相结合，利用预测分布的完整形状信息，生成自适应的预测区间。框架包含四个步骤：训练DGP、选择分布感知分数、校准全局阈值、通过求根定位区间。

### 2.2 关键技术细节

1. **训练分布生成预测器**：
   - 任何输出预测分布 $\hat{P}_x$ 的模型，通过 $M$ 次采样得到样本向量 $\mathbf{\hat{y}}(x)$。
   - 支持的DGP：分位数回归（QR）、MC dropout（MCD）、Bootstrap集成（BE）、深度集成（DE）。

2. **选择非一致性分数**：
   - 分数 $s(y, \hat{\mathbf{y}}(x))$ 量化真实值相对于预测分布的异常程度。
   - 三种分数族：
     - **基于误差**：绝对残差 $s_R = |y - \hat{\mu}|$ 及方差缩放Z分数 $s_Z = |y - \hat{\mu}| / \sigma$。
     - **基于区间违背**：基于预计算的内带（如分位数或最高密度区间HDI）计算 $s_{\text{int}} = \max\{C_{\text{low}} - y, y - C_{\text{up}}\}$，可除以带宽进行缩放。
     - **基于密度**：使用K近邻（KNN）距离作为密度替代，$s_{\text{KNN}} = d_k(y, \hat{\mathbf{y}}) / \tilde{d}$，其中 $\tilde{d}$ 是样本间中位数距离。

3. **校准全局阈值**：
   - 在保留的校准集上计算每个样本的非一致性分数 $\varepsilon_i$，取 $(1-\alpha)$ 经验分位数 $\hat{q}$。
   - 采用在线滑动窗口共形预测，逐步更新 $\hat{q}$ 以应对分布漂移。

4. **通过求根定位区间**：
   - 对于测试点，求解方程 $f(y) = s(y, \mathbf{\hat{y}}_t) - \hat{q} = 0$ 的根。
   - 使用“bracketing + bisection”算法：以预测中位数为锚点，沿几何网格检测符号变化，对最外层符号变化进行二分法求精。
   - 该数值反演方法对分数不可逆的情况（如密度分数、多模态分布）鲁棒，且自动处理负 $\hat{q}$ 的情况。

### 2.3 修正的Winkler分数（MMW）
- 经典Winkler分数 $W_i = \delta_i + e_i$ 结合区间宽度和覆盖缺失惩罚。本文提出欠覆盖惩罚因子 $P_{uc} = e^{\kappa \rho}$，其中 $\rho = \Delta C / (1 - \hat{C})$，$\Delta C = \max(0, C_a - \hat{C})$。
- MMW = $\frac{1}{N}\sum (\delta_i + P_{uc} e_i)$，在覆盖低于可接受水平时急剧增加惩罚，强调欠覆盖的危害。

## 6. 论文的主要结论与发现

1. **效率取决于DGP的不确定性与测试数据对齐**：
   - 当不确定性以偶然异方差为主时，QR + 自适应分数效果最好（最窄区间，低MMW）。
   - 当存在分布偏移（认知不确定性）时，MCD（产生与模型敏感性相关的局部分散）优于QR，因QR缺乏认知信号。
2. **共形校准无法弥补基础预测器的缺陷**：
   - 若预测分布不反映真实不确定性（如QR在偏移下不增加宽度），即使经过校准，区间也不会有效变窄，或需要过度扩大以维持覆盖，导致MMW升高。
3. **自适应分数在信息对齐时改善效率**：
   - 在合适条件下（如Solar数据中QR+QIS/KNN），MMW从0.36降至0.22-0.23。
   - 反之（如MCD在数据上产生不准确分散），自适应分数可能降低覆盖或增加MMW。
4. **MMW分数作为选择标准**：在达到可接受覆盖的方法中，选择MMW最低的组合（结合宽度和欠覆盖惩罚）。

## 7. 优点

1. **统一框架**：将多种现有共形预测混合方法整合到单一体系下，便于比较和选择。
2. **模块化设计**：预测器、分数、校准策略可独立替换，真正的“即插即用”。
3. **数值反演**：分数无关的求根算法，无需为每个分数推导解析逆，处理非单调、多模态分数。
4. **新评估指标MMW**：强调欠覆盖，符合高风险应用中对安全性的要求，比PICP+PINAW更综合。
5. **严谨实验**：覆盖多种不确定性场景，使用多种模型和数据集，结果有统计量支持。
6. **实践指导**：提出简单选择规则：先筛选达到覆盖阈值的方法，再选MMW最低的，并给出针对不同不确定性类型的推荐。
