## 摘要
大语言模型遭受缓慢的自回归推理。虽然自推测解码加速了这一过程，但其效率受到静态配置（如固定的退出层和推测长度）的阻碍。我们将这一优化问题重新定义为马尔可夫决策过程，并提出了LEDE，一个使用离线强化学习的框架。LEDE学习一个策略，根据每一步生成序列的局部上下文动态选择最优的退出层和推测长度，从而平衡计算成本和草稿质量。在Llama-2和Llama-3模型上的全面评估表明，与自回归解码相比，LEDE实现了高达2.0倍到2.7倍的加速，并且与静态推测基线相比，提供了额外的17%加速。

## Abstract
Large Language Models suffer from slow autoregressive inference. While self-speculative decoding accelerates this process, its efficiency is hampered by static configurations like fixed exit layers and speculation lengths. We reframe this optimization as a \textbf{Markov Decision Process} and propose \textbf{LEDE}, a framework that uses offline reinforcement learning. LEDE learns a policy to dynamically select the optimal exit layer and speculation length based on the local context of the generated sequence at each step, balancing computational cost and draft quality. Comprehensive evaluations on Llama-2 and Llama-3 models show LEDE achieves up to a $2.0\times$$\sim$$2.7\times$ speedup over autoregressive decoding and and provides an additional 17\% speedup over the static speculative baselines.

# 论文总结：Experience-Driven Dynamic Exits for LLMs with Reinforcement Learning

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：大型语言模型（LLMs）推理时，自回归解码每生成一个 token 都需要通过全部 Transformer 层，导致延迟高。自推测解码（Self-Speculative Decoding, SSD）利用模型浅层作为草稿模型，并行生成多个候选 token 再由完整模型验证，可加速推理。
- **问题**：现有 SSD 方法采用**静态配置**（固定的退出层和固定的推测长度），无法适应不同 token 的预测难度，导致效率低下。简单启发式方法（如基于置信度阈值）也因预先设定的规则而难以最优平衡计算成本与草稿质量。
- **核心挑战**：如何根据当前生成的序列上下文动态调整退出深度和草稿长度，实现自适应加速。
- **论文目标**：将动态控制问题建模为马尔可夫决策过程（MDP），利用离线强化学习（Offline RL）学习一个策略，在每一步动态选择最优退出层和推测长度，从而显著提升推理速度。

## 2. 方法论：核心思想、关键技术细节与算法流程

- **核心思想**：将 SSD 中的退出层选择与推测长度决策统一为一个序贯决策问题，通过离线强化学习从历史经验中学习最优策略，替代固定规则。
- **关键技术细节**：
  - **MDP 建模**：
    - **状态 (State)**：在每个候选层 \(l\)，从模型内部隐状态提取特征：**Token Confidence**（top-k 平均负对数概率，k=5）、**Token Entropy**（分布熵）、**Average Trace of Token Confidence**（过去 T=20 步 top-1 概率的移动平均）。
    - **动作 (Action)**：在每个候选层，策略输出 `continue`（继续向更深层）或 `exit`（选择当前层作为退出层）。若到达最后一个候选层则强制退出。
    - **奖励 (Reward)**：若退出且草稿 token 被接受：\(1 + \frac{L - l^*}{L}\)；若退出但草稿被拒绝：\(-2 + \frac{L - l^*}{L}\)；若选择继续：\(-0.01\)。鼓励较早退出且被接受，惩罚错误退出。
  - **离线强化学习算法**：采用 Deep Q-Network (DQN)，使用小 MLP 作为 Q 网络。训练前先通过探索收集经验存入回放缓冲池（Replay Buffer）。训练时使用 n-step TD 误差（n=3）更新 Q 网络，目标网络定期复制参数。
  - **双重动态控制**：
    - **动态退出层**：在每步生成时，从浅层到深层逐层评估状态，由策略决定在哪个层退出。
    - **自适应推测长度**：选定退出层后，连续生成草稿 token，每生成一个后重新评估状态：若策略输出 `exit` 则继续草稿，若输出 `continue` 则立即终止草稿并进入验证阶段。
- **算法流程**（文字描述）：
  1. **离线训练阶段**：使用带噪声层的 Q 网络与环境交互（初始探索200 episodes），收集经验元组到回放缓冲池。然后从缓冲池中采样 minibatch（32），计算 n-step TD 误差，最小化均方误差损失更新网络。目标网络每 512 步复制一次。
  2. **在线推理阶段**：对当前生成步骤，从第1层开始，每到达一个候选层，提取状态输入训练好的 Q 网络，选择 `exit` 动作则确定退出层 \(l^*\)；否则继续下一层。随后从 \(l^*\) 开始递进式生成草稿，每生成一个 token 后再次用策略判断：若 `exit` 则继续生成，若 `continue` 则终止草稿，将生成的序列提交给完整模型验证。
