## 摘要
数据混合选择对于大语言模型预训练至关重要。现有方法如RegMix通过在小型代理运行上拟合回归模型来选择单一的静态混合。我们提出RegMix-D，作为RegMix的简单扩展以实现动态混合。我们的关键观察是代理运行不仅产生端点损失，还产生完整的损失轨迹，这些轨迹可以用于进一步改进数据混合。通过在这些轨迹上训练回归模型，我们可以预测多个训练阶段的最优混合。RegMix-D支持两种部署模式：一种离线变体在目标训练之前生成完整的混合计划，另一种在线变体在训练过程中使用观察到的损失自适应调整混合。在Pile数据集上以1B参数目标模型进行的25B token实验表明，RegMix-D在13个下游任务中持续优于RegMix和DoReMi，同时保持代理高效性：即使仅使用128个代理模型（占RegMix代理计算预算的25%），它也超过了RegMix。

## Abstract
Data mixture selection is critical for Large Language Model pretraining. Existing methods such as RegMix select a single static mixture by fitting a regression model on small-scale proxy runs. We propose RegMix-D, a simple extension of RegMix to dynamic mixing. Our key observation is that proxy runs produce not only endpoint losses, but also full loss trajectories, which can be used to further improve data mixture. By training regression model on these trajectories, we can predict optimal mixtures at multiple training stages. RegMix-D supports two deployment modes: an offline variant that generates a complete mixture schedule before target training, and an online variant that adapts the mixture during training using observed loss. Experiments on 25B tokens of the Pile dataset with a 1B parameter target model show that RegMix-D consistently improves over RegMix and DoReMi across 13 downstream tasks while remaining proxy-efficient: it surpasses RegMix even with only 128 proxy models (25% of RegMix's proxy compute budget).

## 2. 论文提出的方法论：核心思想、关键技术细节
*   **核心思想**：扩展 RegMix 的回归框架。在 RegMix 训练代理模型时，不仅记录最终损失，还记录其完整的**损失曲线**。然后，训练一个新的回归模型（同样使用 LightGBM），该模型能够根据当前的训练步骤、当前混合比例和当前损失，预测下一个时间步的损失。利用该模型，可以搜索出在训练各阶段能使预测损失最小的最优混合比例。
*   **关键技术细节**：
    *   **轨迹条件回归**：
        1.  在代理训练中，设定 `N` 个切换点，将训练分为 `N+1` 个间隔。对于每个代理模型，记录每个间隔结束时的验证损失，形成一条损失轨迹。
        2.  构建训练数据对：输入为 `(当前步骤 tℓ，当前混合 m(i)，当前损失 ℓ(i)ℓ)`，目标是 `(下一间隔的损失 ℓ(i)ℓ+1)`。这训练了回归模型 `fN`，使其能学习“状态 -> 未来损失”的局部转移。
    *   **两种部署模式**：
        *   **离线模式**：在目标训练开始前，利用 `fN` 递归地生成完整的混合调度方案。从初始混合 `mhuman` 开始，在第一个切换点，搜索使 `fN` 预测损失最小的混合 `m*1`，并将 `fN` 的输出作为下一个切换点的输入，以此类推。
        *   **在线模式**：在目标训练过程中，当达到切换点时，使用**目标模型本身**在该点的真实验证损失（经过跨尺度校正）作为 `fN` 的输入，来查询当前步骤下的最优混合。相较于离线模式，此模式更准确，因为它基于真实观察而非递归预测。
    *   **跨尺度损失校正**：由于代理模型（1M参数）和目标模型（1B参数）的损失尺度不同，论文提出一种幂律校正方法：`˜ℓ = ℓ̂ · (Pt/Pp)^β`，其中 `β` 是一个小的正数（实验中设为0.05），用于将目标模型的损失映射到代理模型尺度，以便 `fN` 能够正确地理解和预测。
