## 摘要
神经控制微分方程（NCDE）为时间序列预测提供了一种强大的连续时间框架，但标准的基于图的扩展通常纯粹从数据中学习空间结构，即使在有向图结构先验已知的情况下也是如此。我们提出了有信息引导的神经控制微分方程（INDEQS），这是一种基于图的NCDE预测方法，能够在不同的架构位置融入有向图的先验知识。INDEQS将图节点间隐藏状态的内部混合与向量场和控制之间的外部混合分离，并提供了轻量级的图约束变体和更具表达力的变体，后者通过自适应图卷积从数据中学习额外的图连接。为了系统地研究图信息在预测中何时有益，我们设计了有向图上的连续平流模拟，生成具有已知真实流结构的合成时空数据集。然后，我们在两个真实世界任务上评估INDEQS：水文网络上的河流流量预测和PeMS08上的交通流量预测。在这些合成和真实基准测试中，与具有可比参数数量的无信息NCDE相比，外部信息始终能改善平均绝对误差，尤其是在较大图上，而内部信息在需要严格遵循已知邻接关系时提供了更参数高效的替代方案。离散卷积解码器与连续时间解码器的比较进一步表明，连续解码器在真实世界任务中能提供更好的精度和更大的时间灵活性。INDEQS的实现和平流模拟可在https://github.com/Mitchi1/indeqs获取。

## Abstract
Neural Controlled Differential Equations (NCDE) provide a powerful continuous-time framework for forecasting time series, but standard graph-based extensions typically learn spatial structure purely from data, even in settings where a directed graph structure is known a priori. We introduce Informed Neural controlled Differential EQuationS (INDEQS), a graph-based NCDE forecasting method that incorporates prior knowledge of a directed graph at distinct architectural positions. INDEQS separates inner mixing of hidden states across graph nodes from outer mixing between vector field and control, and offers both a lightweight graph-constrained variant and a more expressive variant, learning additional graph connections from data via adaptive graph convolutions. To systematically study when graph informedness is beneficial in forecasting, we devise a continuous advection simulation on directed graphs, yielding synthetic spatio-temporal datasets with known ground-truth flow structure. We then evaluate INDEQS on two real-world tasks: river discharge forecasting on a hydrological network and traffic flow prediction on PeMS08. Across these synthetic and real-world benchmarks, outer informedness consistently improves mean absolute error over an uninformed NCDE with comparable parameter count, particularly on larger graphs, while inner informedness offers a more parameter-efficient alternative when strict adherence to a known adjacency is desired. A comparison of discrete convolutional and continuous-time decoders further shows that continuous decoders yield better accuracy and greater temporal flexibility on real-world tasks. An implementation of INDEQS and the advection simulation is available at https://github.com/Mitchi1/indeqs.

## 2. 方法论

### 2.1 核心思想
INDEQS 基于双层 NCDE 架构（一个 NCDE 编码观测路径，另一个 NCDE 处理隐藏状态）。关键创新在于将**先验有向图邻接矩阵**注入到矢场的两个不同位置：
- **外部位置**：将图矩阵直接作用于控制路径的导数（`dH/dt`）上，然后再与内部映射相乘。这相当于在“读取控制信息”时，利用图拓扑来加权不同节点的控制信号。
- **内部位置**：在图卷积操作中使用已知的硬编码邻接矩阵（而非可学习的节点嵌入）来驱动隐藏状态 Z 的更新，从而严格限制信息只能在有向边连接的节点间传播。

### 2.2 关键技术细节

#### 2.2.1 基础 NCDE 架构
标准 STG‑NCDE（Choi et al., 2022）由一个 NCDE 编码观测路径 X(t) 得到隐藏状态 H(t)，再由另一个 NCDE（其矢场 g）将 H(t) 映射为最终隐藏状态 Z(t)：
```
H(T) = H(0) + ∫ f(H(t)) dX(t)/dt dt
Z(T) = Z(0) + ∫ g(Z(t)) dH(t)/dt dt
```
其中 f 为逐节点处理（无空间混合），g 内部使用**自适应图卷积（AGC）**学习空间依赖（通过可学习节点嵌入生成动态邻接矩阵）。

#### 2.2.2 INDEQS outer（外部信息）
在 g 的输出端引入先验图矩阵 A_transpose_outer：
```
Z(T) = Z(0) + ∫ g(Z(t)) * A_transpose_outer * dH(t)/dt dt
```
其中 g 保持 ABC 结构（AGC），但此时外部信息提供了一个“已知的拓扑滤波”，限制控制信号在节点间的传播方式。默认 A_transpose_outer 为节点转移矩阵的转置（反映上游到下游的方向）。当 A_transpose_outer 设为单位阵时，退化为标准 STG‑NCDE。

#### 2.2.3 INDEQS inner（内部信息）
保留外层为单位矩阵（无外部图约束），但将 g 内部的 AGC 替换为固定邻接矩阵：
```
g2(B) = A_transpose_inner * B
```
从而彻底消除可学习的图参数，使得隐藏状态更新严格遵循已知图连接。参数更少（省去了节点嵌入），但无法学习图外的额外连接。

#### 2.2.4 高阶邻接
对于需要多跳传播的任务，可将邻接矩阵替换为 k 次幂之和：
```
A_k = Σ (A^i) for i=0..k
```
从而允许信息沿多步路径传播。

#### 2.2.5 连续解码器
除了默认的卷积解码器（对 Z(T) 做线性映射生成未来预测），还提出了两种连续解码器：
- **NODE g 解码器**：以 Z(T) 为初值，用同构的矢场 g 继续演化解 NODE，在任意未来时间点输出隐藏状态，再映射到预测值。
- **NODE ctrl 解码器**：额外学习一个 NODE 来外推控制路径 X(t)，然后用该外推控制驱动原 NCDE 继续演化，从而得到未来隐状态。

### 2.3 算法流程（文字描述）
1. 输入：时间序列观测 { (t_i, x_i) }，以及已知有向图邻接矩阵 A。
2. 用 Hermite 三次样条插值得到连续控制路径 X(t)。
3. 定义 NCDE 矢场 f（逐节点 MLP）和 g（内部含图操作，位置依变体而定）。
4. 从初始隐藏状态 H(0)、Z(0) 开始，在 [0, T] 上解耦合 NCDE，得到 Z(T)。
5. 通过解码器（卷积或连续）生成未来预测值。
6. 使用 MAE 损失优化所有参数。

## 6. 主要结论与发现

1. **外知情（INDEQS outer）一致优于无信息 NCDE**：在合成数据和两个真实任务中，外知情的 MAE 均低于 STG‑NCDE（参数数相同），且随着图节点数增加，优势愈发明显。
2. **内知情（INDEQS inner）在特定条件下有效**：对于中等以上规模的图（节点数 ≥ 16），内知情的 MAE 低于 STG‑NCDE，且参数更少、训练更快（约 24%）。但在小图上不如自适应 AGC。
3. **高阶邻接提升性能**：在河流数据上，随着邻接矩阵幂次增加（k 从 1 到 12），外知情和内知情的 MAE 均持续下降，表明多跳信息传播的重要性。
4. **连续解码器优于离散卷积解码器**：在河流和交通任务上，NODE g 和 NODE ctrl 解码器的 MAE 均低于卷积解码器，但训练时间更长。在合成数据上连续解码器表现较差（可能因任务特征）。
5. **图信息在含噪声真实场景仍有增益**：即使交通流（含较多随机因素）上，INDEQS outer 仍微弱优于 STG‑NCDE，表明道路网络拓扑的有效性。

## 7. 优点

- **架构创新**：首次在 NCDE 中明确分离“内部混合”和“外部混合”位置，并分别注入图信息，提供了清晰的设计空间。
- **两种变体满足不同需求**：外知情兼顾先验与数据自适应能力；内知情提供严格约束且参数高效，适合确信图结构完全可靠的场景。
- **合成数据设计精巧**：基于连续边平流模拟生成具有已知因果流的时空数据，为控制实验提供了理想测试床。
- **全面消融**：涵盖了邻接矩阵幂次、内/外位置组合、解码器类型、求解器步长等多维消融，结果稳健。
- **开源代码与数据**：全部代码和模拟数据公开，可复现。
- **详实的比较**：不仅对比连续时间模型，还纳入了离散时间 SOTA 方法（DCRNN、GWaveNet），尽管后者在某些任务上表现更好，但展示了 INDEQS 作为连续时间模型的竞争力。

## 8. 不足与局限

- **依赖已知图结构**：若图不可靠或不存在，INDEQS 无法使用（此时退化为 STG‑NCDE）。方法假设图能反映真实因果传播，但现实场景中可能难以区分“物理连接”和“空间邻近”。
- **评估范围有限**：仅测试了 3 种类型的数据集（合成河网、真实河网、高速公路网），未涵盖其他领域（如流行病传播、物理模拟等）；图规模最大仅 170 节点，大图（如千级以上）性能未验证。
- **离散时间基线在某些任务上更强**：在河流任务上，GWaveNet 的 MAE（4.95）明显优于 INDEQS（5.01–5.34），说明连续时间模型并非总是最优点。
- **连续解码器的适用性局限**：在合成数据上连续解码器远差于卷积解码器，原因未被深入解释，可能是任务特性（平流需要精确对齐时间步）。
- **计算开销**：外知情相对于无信息 NCDE 增加有限，但连续解码器（尤其 NODE ctrl）需要额外 ODE 求解，训练时间增加数倍，且需要调节控制路径损失权重。
- **未探究动态图**：本文限定于静态图，而现实中的图可能随时间变化（如交通路况），INDEQS 未扩展至动态图场景。
- **超参数敏感性未充分讨论**：例如求解器步长和类型对性能有明显影响（图 11），但论文主要使用固定配置（RK4 步长 1），可能不是最优。
