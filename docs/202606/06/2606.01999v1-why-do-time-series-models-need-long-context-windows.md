## 摘要
现代深度学习模型用于预测时间序列组时，越来越依赖更长的观测窗口。然而，增加窗口大小的好处通常仅仅被归因于捕获长程依赖关系，而对于全局预测模型如何利用输入观测的广泛讨论一直有限。在本文中，我们表明预测时间序列组涉及两个目标：(i) 生成过程识别（GPI），即推断生成输入序列的具体过程，以及 (ii) 条件预测（CF），即根据输入观测预测未来值。从这个角度来看，最优预测可以被解释为对多个可能的数据生成过程取平均，这些过程根据输入窗口的可能性进行加权。这为长上下文窗口的好处提供了另一种解释：它们减少了在运行过程中对具体是哪个过程生成输入时间序列的不确定性。我们证明，即使对于记忆长度为 $P$ 的过程，输入窗口大小严格大于 $P$ 也是实现最小可达误差所必需的。最后，我们展示了如何将 GPI 和 CF 解耦，可以在不牺牲准确性的情况下提高计算可扩展性。在合成数据和真实世界数据上的实验验证了我们的见解及其在设计预测架构中的相关性。

## Abstract
Modern deep learning models for forecasting groups of time series rely on increasingly longer observation windows. However, the benefit of increasing the window size is often simply attributed to capturing long-range dependencies, and broader discussion on how global forecasting models leverage input observations has been limited. In this paper, we show that forecasting groups of time series involves two objectives: (i) generative process identification (GPI), i.e., inferring the specific process generating the input sequence, and (ii) conditional forecasting (CF), i.e., predicting future values given input observations. From this perspective, optimal predictions can be interpreted as an average over plausible data-generating processes, weighted by their likelihood given the input window. This suggests another explanation for the benefits of long context windows: they reduce the uncertainty about which specific process is generating the input time series during operation. We prove that even for processes with memory length $P$, an input window size strictly larger than $P$ is necessary to achieve the minimum attainable error. Finally, we show how decoupling GPI and CF can improve computational scalability without compromising accuracy. Experiments on synthetic and real-world data validate our insights and their relevance for designing forecasting architectures.

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：现代深度学习时间序列预测模型（尤其是基础模型）越来越依赖更长的观测窗口（context window）。传统观点认为，长窗口的好处在于捕获长程依赖关系（long-range dependencies）。
- **核心问题**：作者质疑这一观点，指出尽管存在长程依赖，但许多实际数据中相关滞后长度（如偏自相关所示）远小于模型选择的最优窗口。那么，长窗口的真正作用是什么？
- **整体含义**：论文提出，全局预测模型（global forecasting model）在预测一组时间序列时，实际上执行了两个隐含目标：
  - **生成过程识别（GPI）**：从输入序列中推断出具体是哪个数据生成过程在产生观测。
  - **条件预测（CF）**：给定过去观测，预测未来值。
- **新解释**：最优预测可视为对所有可能生成过程的期望加权平均，权重由后验概率决定。长窗口的主要作用是**减少关于哪个过程生成数据的不确定性**，而不仅仅是捕获长程依赖。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- 将预测任务形式化为一个贝叶斯后验平均问题。令模型参数为 $\Theta$，输入窗口 $x_{t-W:t}$，目标 $x_t$。在 MSE 损失下，最优预测为：
  \[
  \hat{x}_t = \mathbb{E}[x_t \mid x_{t-W:t}] = \int \mathbb{E}[x_t \mid x_{t-W:t}, \phi] \, p(\phi \mid x_{t-W:t}) \, d\phi
  \]
  其中 $\phi$ 是生成过程的参数（即 GPI），$\mathbb{E}[x_t \mid x_{t-W:t}, \phi]$ 是已知过程后的条件预测（CF）。
- 因此，窗口 $W$ 的作用同时体现在：GPI（通过后验 $p(\phi \mid x_{t-W:t})$）和 CF（通过条件期望）。**即使过程记忆长度 $P$ 很小，GPI 也需要更多观测才能降低 $\phi$ 的不确定性**。

### 关键技术细节
- **理论证明（Theorem 3.2）**：对于阶数为 $P$ 的马尔可夫过程，在合理假设下（Assumption 3.1），**窗口长度 $W > P$ 是达到最小误差的必要条件**。证明通过条件方差分解：当 $W=P$ 时，误差包含一项 $\mathbb{E}[\mathrm{Var}_{\phi}(\mathbb{E}[x_t \mid x_{t-P:t}, \phi] \mid x_{t-P:t})] > 0$，不为零；只有 $W > P$ 才能通过更多观测收缩后验，消除此方差。
- **解耦设计**：作者提出将 GPI 和 CF 分离，设计双模块模型：
  - **G 模块**：接收历史上下文 $x_{t'-C:t'}$（$C$ 长，$t'$ 远早于 $t$），输出嵌入 $e_n$，表征生成过程身份。
  - **F 模块**：接收最近窗口 $x_{t-W:t}$ 和嵌入 $e_n$，进行条件预测。
  - 优势：上下文 $C$ 只需处理一次（可缓存），后续只更新短窗口 $W$ 进行预测，降低推理成本。
- 具体实现：G 可用 PatchTST、线性层等；F 可用 PatchTST、TSMixer、ModernTCN、LRU 等。

### 算法流程（文字说明）
1. 预处理：从目标时间序列的历史中采样一段长度为 $C$ 的上下文序列（非相邻，有间隔 $S$）。
2. 编码：用 G 模块将上下文编码为嵌入向量 $e_n$。
3. 预测：用 F 模块处理最近 $W$ 个观测，并注入 $e_n$（如相加、拼接或作为初始状态），输出未来 $H$ 步预测。
4. 推理时：嵌入 $e_n$ 可固定，每次只用新窗口进行预测，比传统方法每次都要处理 $C+W$ 长度序列更高效。

## 6. 主要结论与发现

1. **长窗口的真正作用**：主要在于降低生成过程的不确定性（GPI），而非仅仅捕获长程依赖。即使过程记忆长度很小，模型也可能需要更长窗口来识别具体过程。
2. **必要窗口长度定理**：对于阶数为 $P$ 的马尔可夫过程，在一般条件下，$W > P$ 是达到最小误差所必需的。
3. **基础模型成本更高**：预训练基础模型相比领域专用全局模型，需要更长的输入窗口才能匹配性能（图 5），因为其预训练分布更广，GPI 更难。
4. **解耦 GPI 和 CF 可提升效率**：分离后的设计（上下文编码 + 短窗口预测）能在几乎不损失准确性的情况下显著降低计算成本（图 7、表 6），尤其在需要多次预测同一时间序列时。
5. **GPI 需要更高模型容量**：线性模型无法很好执行 GPI（图 9、图 11），而深层架构（如 PatchTST）更有效。
