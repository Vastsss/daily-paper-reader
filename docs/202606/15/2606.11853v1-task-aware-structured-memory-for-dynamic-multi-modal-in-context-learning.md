## 摘要
多模态大语言模型依赖于上下文学习进行快速任务适应，但其可扩展性受到有限上下文窗口以及长多模态序列中键值缓存成本增长的严重限制。现有的记忆压缩方法通常依赖于刚性令牌移除或样本相关的重要性估计，这引入了偏差，破坏了语义结构（尤其对于视觉表示），并产生无法适应新查询的静态记忆。我们提出了TASM（任务感知结构化记忆），这是一个无需训练的框架，通过任务感知、结构保持和动态可访问的记忆构建来解决这些限制。TASM采用任务向量引导的压缩，用捕捉演示间共享相关性的任务级方向替换样本特定信号。为了保持底层流形，它通过二分图匹配进行语义感知的令牌合并，在不进行破坏性剪枝的情况下聚合令牌。最后，TASM将记忆组织成一个层次结构，包括紧凑的核心记忆和潜在库，促进查询自适应的动态检索。评估证实，TASM在高度压缩下保持高性能，有效平衡效率与适应性。

## Abstract
Multi-modal large language models (MLLMs) depend on in-context learning (ICL) for rapid task adaptation, but their scalability is severely limited by finite context windows and the growing cost of key-value (KV) caches in long multi-modal sequences. Existing memory compression approaches typically rely on rigid token removal or sample-dependent importance estimation, which introduces bias, disrupts semantic structure, particularly for visual representations, and yields static memories that cannot adapt to new queries. We introduce TASM (Task-Aware Structured Memory), a training-free framework that addresses these limitations through task-aware, structure-preserving, and dynamically accessible memory construction. TASM employs task-vector guided compression to replace sample-specific signals with a task-level direction that captures shared relevance across demonstrations. To preserve the underlying manifold, it applies semantics-aware token merging via bipartite graph matching, aggregating tokens without destructive pruning. Finally, TASM structures memory into a hierarchy comprising a compact Core Memory and a Latent Bank, facilitating query-adaptive dynamic retrieval. Evaluations confirm TASM maintains high performance under heavy compression, effectively balancing efficiency with adaptability.

# 论文总结：Task-Aware Structured Memory for Dynamic Multi-modal In-Context Learning

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：多模态大语言模型（MLLMs）依赖于上下文学习（ICL）进行快速任务适应，但长序列下 KV 缓存的内存和计算成本严重制约可扩展性。现有记忆压缩方法（如硬剪枝、注意力评分）存在三个根本缺陷：
  1. **样本特异性偏差**：基于演示回答的注意力分数会过拟合到特定示例，丢弃对查询重要的上下文。
  2. **拓扑结构破坏**：视觉令牌具有固有的 2D 空间语义结构，硬剪枝会破坏空间关系（如目标定位）。
  3. **静态记忆僵硬**：压缩后的缓存是静态的，无法恢复被丢弃但对复杂查询必要的信息。
- **含义**：需要一种无训练（training-free）、结构保持、动态可访问的记忆压缩框架，以在保持高性能的同时实现高效的长上下文多模态上下文学习。

## 2. 方法论：核心思想、关键技术细节

### 2.1 整体框架（图2）
- 分为两阶段：
  - **离线压缩阶段**：从支持集提取任务向量，对 KV 缓存进行重要性评分和语义感知合并，构建 GPU 驻留的核心记忆（Core Memory）和 CPU 驻留的潜库（Latent Bank）。
  - **在线推理阶段**：根据查询的注意力分布变化（JS 散度）动态决定是否从潜库检索补充令牌，最后通过多头注意力生成。

### 2.2 三项关键技术

1. **任务向量引导的重要性估计**
   - 从少量问答对（演示）中提取任务向量 \(\tau_l = \frac{\mu_{A,l} - \mu_{Q,l}}{\|\mu_{A,l} - \mu_{Q,l}\|_2}\)，表示从问题到答案的变换方向。
   - 通过正交投影计算令牌的任务相关性分数 \(s_{i,l}^{\text{task}} = \text{ReLU}\left( \frac{k_{i,l}^\top \tau_l}{\|k_{i,l}\|} \right) + \gamma \frac{\|v_{i,l}\|_2}{\max_j \|v_{j,l}\|_2}\)。
   - 引入层自适应门控 \(\lambda(l) = \alpha + \beta \cdot \sigma(\kappa \cdot (l/L - 0.5))\)，浅层侧重局部注意力，深层侧重全局任务向量。

2. **语义感知令牌合并**
   - 将高重要性令牌设为“汇点”（sink），低重要性令牌设为“源点”（source），构建二分图进行匹配。
   - 边缘权重：余弦相似度 + 空间约束（仅允许空间邻近的视觉令牌合并，\(\Delta_{\text{win}}=3\)）。
   - 聚合更新：汇点令牌更新为其簇的加权质心，实现信息压缩而非丢弃。

3. **信息几何动态检索**
   - 使用 Jensen-Shannon 散度监测当前查询注意力分布与历史分布之间的变化，当超过阈值 \(\epsilon\) 时触发检索。
   - 从潜库取回 top-\(k\) 相似令牌，临时扩展当前缓存，实现自适应长历史访问。

## 6. 主要结论与发现

- TASM 在几乎所有基准上**超越或持平全上下文**，同时降低约 80% 内存（如 ImageNet-100 从 16264 降至 3485 令牌）。
- 在空间敏感任务（ScreenSpot）和时间任务（YouCook2）上相比 EMLoC 有显著提升（+1.2% 和 +7.5%），证明剪枝破坏拓扑结构、任务向量+软合并有效。
- 在 V-NIAH 极长检索任务中以 45.5%（200 张图像）远超 EMLoC 的 18.9%，证明动态检索克服了“记忆僵硬”。
- 任务向量可零样本迁移（目标类与源类不同仅有 0.8% 下降），表明其捕获了“问题→答案”的抽象变换。
- 系统效率：准线性缩放，32k 上下文时仍保持 43.8 tok/s，内存仅 11.5 GB。

## 7. 优点

- **无需训练**：可直接应用于现有冻结 MLLM，降低部署成本。
- **结构保持**：软合并保留视觉令牌的空间拓扑，避免破坏性剪枝。
- **动态适应**：基于 JS 散度的检索仅在需要时激活，平衡效率与准确性。
- **通用性强**：在多种 MLLM 架构（Qwen、LLaVA、InternVL）和任务类型上效果一致。
- **轻量高效**：任务向量提取仅需少量演示（4-shot 即可饱和），潜库可通过 CPU 卸载减少 GPU 压力。

## 8. 不足与局限

- **额外推理开销**：虽然解码吞吐接近 EMLoC，但预填充阶段因图匹配引入一定延迟（+35ms @32k），且需维护潜库（增加 1.3 GB 内存）。
- **依赖演示质量**：任务向量质量受少量问答对影响，在极端稀疏或不平衡演示下可能不够鲁棒（论文仅验证了 1–16 shot，未测试更少或噪声情形）。
- **未与可训练方法对比**：论文只对比了无训练压缩方法，未纳入可训练的稀疏注意力或蒸馏加速方法。
- **阈值敏感性**：JS 阈值 \(\delta\) 需要二参调优，不同任务或模型可能最优值不同，论文仅在一个基准上分析了灵敏度。
- **应用限制**：当前仅用于 MLLM 的 ICL 场景，未测试纯文本 LLM 或其他模态（如音频）；动态检索的机制在非常长的上下文中可能仍有累积误差风险。
