## 摘要
智能体大语言模型系统已展现出强大的能力。然而，它们对语言作为通用接口的依赖，从根本上限制了其在许多现实问题中的适用性，特别是在科学领域，这些领域已经开发了特定领域的基座模型，用以处理自然语言之外的专业任务。在这项工作中，我们介绍了 Eywa，这是一个异构智能体框架，旨在将以语言为中心的系统扩展到更广泛的科学基座模型类别。Eywa 的核心思想是为特定领域的基座模型增强基于语言模型的推理接口，使语言模型能够引导非语言数据模态的推理。这种设计允许通常针对专业数据和任务进行优化的预测性基座模型，参与到智能体系统内更高层级的推理和决策过程中。Eywa 可以作为单智能体流水线（EywaAgent）的即插即用替代方案，或者通过将传统智能体替换为专业智能体（EywaMAS）来集成到现有的多智能体系统中。我们进一步研究了一种基于规划的编排框架，其中规划器动态协调传统智能体和 Eywa 智能体，以解决跨异构数据模态的复杂任务（EywaOrchestra）。我们在涵盖物理、生命和社会科学的多个科学领域对 Eywa 进行了评估。实验结果表明，Eywa 提高了处理结构化和特定领域数据任务的性能，同时通过与专业基座模型的有效协作，减少了对基于语言推理的依赖。

## Abstract
Agentic large language model systems have demonstrated strong capabilities. However, their reliance on language as the universal interface fundamentally limits their applicability to many real-world problems, especially in scientific domains where domain-specific foundation models have been developed to address specialized tasks beyond natural language. In this work, we introduce Eywa, a heterogeneous agentic framework designed to extend language-centric systems to a broader class of scientific foundation models. The key idea of Eywa is to augment domain-specific foundation models with a language-model-based reasoning interface, enabling language models to guide inference over non-linguistic data modalities. This design allows predictive foundation models, which are typically optimized for specialized data and tasks, to participate in higher-level reasoning and decision-making processes within agentic systems. Eywa can serve as a drop-in replacement for a single-agent pipeline (EywaAgent) or be integrated into existing multi-agent systems by replacing traditional agents with specialized agents (EywaMAS). We further investigate a planning-based orchestration framework in which a planner dynamically coordinates traditional agents and Eywa agents to solve complex tasks across heterogeneous data modalities (EywaOrchestra). We evaluate Eywa across a diverse set of scientific domains spanning physical, life, and social sciences. Experimental results demonstrate that Eywa improves performance on tasks involving structured and domain-specific data, while reducing reliance on language-based reasoning through effective collaboration with specialized foundation models.

### 2. 方法论：核心思想与关键技术
*   **核心思想**：通过“FM-LLM Tsaheylu Bond”接口，将领域基座模型包装成具备语言推理能力的智能体。
*   **关键技术细节**：
    *   **查询编译器（Query Compiler, $\phi_k$）**：将 LLM 的内部推理状态转换为 FM 可执行的结构化配置参数。

    *   **模型上下文协议（MCP）**：利用 Anthropic 提出的 MCP 标准化接口，实现 LLM 与外部 FM 服务的无缝通信。
*   **三种实例化形式**：
    1.  **EywaAgent**：单智能体模式，LLM 动态决定是进行语言推理还是调用 FM 进行专业计算。
    2.  **EywaMAS**：多智能体系统，将现有的语言智能体替换为 EywaAgent，实现跨领域的专家协作。
    3.  **EywaOrchestra**：动态编排模式，由一个“指挥家”（Conductor）根据任务需求，动态配置智能体角色、选择 FM 模型及通信拓扑。

### 5. 实验数量与充分性
*   **实验规模**：在 EywaBench-V1 的 200 个任务实例上进行了测试，涵盖 9 个子领域。
*   **消融实验**：进行了丰富的消融研究，包括不同 LLM 后端（GPT-4.1-nano 到 GPT-5-mini）、采样温度敏感性、提示词设计（CoT, ReAct）以及 FM 自身的参数影响。
*   **充分性评价**：实验设计较为全面，不仅关注准确率（Utility），还系统对比了 Token 消耗和执行时间，证明了框架在性能与成本之间的平衡。

### 6. 主要结论与发现
*   **性能提升**：EywaAgent 相比纯语言智能体，在科学任务上的效用（Utility）提升了约 7%，同时 **Token 消耗减少了约 30%**，执行速度提升了 10%。
*   **异构优势**：单纯增加 LLM 的异构性（如 MoA）在科学任务上效果有限，引入跨模态的 FM 协作才是提升性能的关键。
*   **动态编排**：EywaOrchestra 能够自动选择最优配置，在无需人工干预的情况下达到接近专家设计的性能，且成本更低。

### 7. 优点
*   **模态原生协作**：避免了将复杂科学数据强行转化为文本的“信息瓶颈”，保留了数据的结构化特征。
*   **高效率**：通过将计算外包给轻量级的专业 FM，显著降低了 LLM 的推理负担和 Token 成本。
*   **理论支撑**：论文提供了信息论分析，证明了语言接口在处理非语言数据时的固有风险（Bayes Risk Gap）。
*   **即插即用**：基于 MCP 协议的设计使其易于集成到现有的智能体工作流中。

### 8. 不足与局限
*   **样本规模**：EywaBench-V1 目前仅包含 200 个样本，对于一个旨在成为“基座”级别的框架来说，样本量仍有待扩充。
*   **FM 依赖性**：系统的最终表现高度依赖于所选 FM 的质量，如果 FM 本身在某些极端科学场景下失效，LLM 很难通过语言推理进行补救。
*   **领域覆盖**：目前主要验证了时间序列和表格数据，对于分子结构、蛋白质折叠等更复杂的科学模态，尚未在文中进行大规模实验验证。
