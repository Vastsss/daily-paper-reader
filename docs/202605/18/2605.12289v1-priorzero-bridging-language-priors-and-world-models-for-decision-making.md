## 摘要
利用大语言模型（LLMs）丰富的世界知识来增强强化学习（RL）智能体，是通往通用智能的一条充满前景的路径。然而，一种根本性的先验-动力学不匹配阻碍了现有方法：静态的 LLM 知识无法直接适应长时程任务中复杂的转移动力学。将 LLM 先验作为固定策略使用会限制探索的多样性，因为先验对特定环境的动力学是盲目的；而端到端微调则面临优化不稳定和信用分配问题。为了弥补这一差距，我们提出了 PriorZero，这是一个统一的框架，通过解耦的展开-训练设计，将源自 LLM 的概念先验集成到基于世界模型的规划中。在展开阶段，一种新颖的根节点先验注入机制仅在蒙特卡洛树搜索（MCTS）的根节点引入 LLM 先验，使搜索集中在语义上有前景的动作上，同时保留世界模型的深度前瞻能力。在训练阶段，PriorZero 将世界模型学习与 LLM 适配解耦：世界模型在交互数据上不断优化，以共同提升其动力学、策略和价值预测；随后利用其价值估计，通过交替优化为稳定的 LLM 微调提供细粒度的信用分配信号。在包括 Jericho 文本冒险游戏和 BabyAI 指令遵循网格世界任务在内的多个基准测试上的实验表明，PriorZero 持续提升了探索效率和渐近性能，为 LLM 赋能的决策制定建立了一个极具前景的框架。我们的代码可在 https://github.com/opendilab/LightZero 获取。

## Abstract
Leveraging the rich world knowledge of Large Language Models (LLMs) to enhance Reinforcement Learning (RL) agents offers a promising path toward general intelligence. However, a fundamental prior-dynamics mismatch hinders existing approaches: static LLM knowledge cannot directly adapt to the complex transition dynamics of long-horizon tasks. Using LLM priors as fixed policies limits exploration diversity, as the prior is blind to environment-specific dynamics; while end-to-end fine-tuning suffers from optimization instability and credit assignment issues. To bridge this gap, we propose PriorZero, a unified framework that integrates LLM-derived conceptual priors into world-model-based planning through a decoupled rollout-training design. During rollout, a novel root-prior injection mechanism incorporates LLM priors exclusively at the root node of Monte Carlo Tree Search (MCTS), focusing search on semantically promising actions while preserving the world model's deep lookahead capability. During training, PriorZero decouples world-model learning from LLM adaptation: the world model is continuously refined on interaction data to jointly improve its dynamics, policy, and value predictions, its value estimates are then leveraged to provide fine-grained credit assignment signals for stable LLM fine-tuning via alternating optimization. Experiments across diverse benchmarks, including text-based adventure games in Jericho and instruction-following gridworld tasks in BabyAI, demonstrate that PriorZero consistently improves both exploration efficiency and asymptotic performance, establishing a promising framework for LLM-empowered decision-making. Our code is available at https://github.com/opendilab/LightZero.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：现有的 LLM 增强 RL 方法存在“先验-动力学不匹配”（Prior-Dynamics Mismatch）的问题。
    *   LLM 拥有丰富的常识，但对特定环境的转移动力学（Transitions）是盲目的。
    *   直接将 LLM 作为策略（Policy）会导致决策短视；端到端微调 LLM 则面临高方差的信用分配（Credit Assignment）和优化不稳定的挑战。
*   **研究动机**：如何既能利用 LLM 的语义引导来加速探索，又能利用世界模型来处理环境特定的动力学建模和长时程规划。

### 2. 方法论：核心思想与关键技术
PriorZero 采用了**解耦的生成-训练（Rollout-Training）架构**：
*   **根节点先验注入（Root-Prior Injection）**：
    *   在蒙特卡洛树搜索（MCTS）中，LLM 的语义先验仅注入到**根节点**。
    *   **公式化表达**：根节点的动作选择概率由 LLM 先验和世界模型策略加权融合而成；而搜索树的内部节点则完全依赖世界模型进行深度前瞻。
    *   **结构化先验提取**：采用“先分析再决策”的思维链（CoT）提示词模板，减少 LLM 的合理化偏差。
*   **交替强化微调（Alternating Reinforcement Fine-Tuning）**：
    *   **解耦优化**：世界模型和 LLM 按照非对称的交替频率更新（如 10:1）。
    *   **低方差信用分配**：利用世界模型的价值估计（Value Estimates）构建 n 步自举（Bootstrapped）TD 优势信号（Advantage），作为 LLM PPO 微调的监督信号。这避免了直接使用稀疏环境奖励带来的高方差问题。

*   **对比方法（五大范式）**：

    *   **P4 (Text Encoder)**：将 LLM 作为冻结的文本特征提取器。

### 6. 主要结论与发现
*   **性能领先**：PriorZero 在所有测试环境中均显著优于纯世界模型（UniZero）和其他 LLM 迁移范式。
*   **探索效率**：在稀疏奖励任务（如 *Zork1*）中，LLM 先验能有效引导 MCTS 避开无意义的循环探索，显著加快收敛速度。
*   **微调稳定性**：通过世界模型提供的价值信号进行交替微调，比直接进行端到端 RLFT 更稳定，且能持续提升 LLM 的决策质量。
*   **规模效应**：增大 LLM 规模（从 3B 到 7B）能提供更强的语义引导，尤其在复杂任务中表现更优。

### 7. 优点
*   **架构优雅**：通过“根节点注入”巧妙解决了 LLM 语义空间与世界模型潜空间在深度搜索时的对齐难题。
*   **解决痛点**：针对 RLFT 中常见的信用分配难题，提出了利用成熟的世界模型作为“评论家”（Critic）的方案。
*   **通用性强**：在纯文本（Jericho）和具身智能模拟环境（BabyAI）中均表现出色。

### 8. 不足与局限
*   **动作空间限制**：目前主要针对离散动作空间（Discrete Action Spaces）设计，尚未在连续控制任务中验证。
*   **计算开销**：虽然通过根节点注入降低了频率，但推理时调用 LLM 仍比传统 RL 模型更耗时。
*   **依赖性**：LLM 的微调效果高度依赖于世界模型价值估计的准确性，因此需要较长的预热期（Warm-up）。
