## 摘要
时间序列中的因果发现越来越多地使用非线性机器学习模型进行，但由此产生的因果关系几乎总是被总结为标量边分数。我们认为这种做法掩盖了非线性自回归模型真正学到的对象：一个状态依赖的函数，其效应随状态、幅度和背景而变化。我们为加性的、可分解贡献的结构形式化了函数值因果影响，并表明标量因果分数构成了一个严重的信息瓶颈，混淆了状态间变异与状态内残差噪声。以神经加性向量自回归为代表架构，我们引入了一个基于个体条件期望的实用框架，可以直接从训练好的模型估计因果响应函数。通过受控的合成实验，我们证明了具有不可区分标量分数的边可以展现定性不同的函数行为，包括单调、阈值、饱和和变符号效应。一个关于民主发展的应用案例进一步表明，函数值分析揭示了以分数为中心的方法系统性遗漏的特定状态和非对称因果结构。

## Abstract
Causal discovery in time series is increasingly performed using nonlinear machine-learning models, yet the resulting causal relationships are almost always summarized by scalar edge scores. We argue that this practice obscures the true object learned by nonlinear autoregressive models: a state-dependent function whose effect varies across regimes, magnitudes, and contexts. We formalize function-valued causal influence for additive, contribution-decomposable architectures and show that scalar causal scores constitute a severe information bottleneck, conflating between-state variation with within-state residual noise. Using Neural Additive Vector Autoregression as a representative architecture, we introduce a practical framework based on Individual Conditional Expectation for estimating causal response functions directly from trained models. Through controlled synthetic experiments, we demonstrate that edges with indistinguishable scalar scores can exhibit qualitatively different functional behaviors, including monotonic, thresholded, saturating, and sign-changing effects. An applied case study on democratic development further shows that function-valued analysis reveals regime-specific and asymmetric causal structure systematically missed by score-centric approaches.

#### 1. 核心问题与整体含义（研究动机与背景）

*   **核心观察**：当前对非线性时间序列进行因果发现时，研究者通常使用非线性机器学习模型（如神经网络）。然而，输出结果几乎总是被简化成一个单一的**标量因果分数**来衡量每条有向边的“强度”。
*   **核心问题**：作者指出这种实践存在根本性的**“表示不匹配”** 问题。非线性模型学习的本质是一个**状态依赖函数**（state-dependent function），其效应会随系统状态（目标变量的不同取值、不同发展阶段）而变化。用一个标量分数来概括这个复杂的函数，会丢失阈值效应、饱和效应、符号翻转等关键结构信息。
*   **整体含义**：标量因果分数构成了一个严重的**信息瓶颈**，它混淆了真正的状态间变化（函数值的变化）与状态内残差噪声。这导致两个具有截然不同因果机制（例如单调和阈值效应）的边可能拥有几乎相同的标量分数，从而误导实际应用（尤其在政治学、经济学等注重机制解释的领域）。

#### 2. 论文提出的方法论

*   **核心思想**：放弃使用标量分数，转而将每个有向边的因果影响定义为一个**函数值的对象**，即**因果响应函数**。这个函数刻画了源变量在不同取值下对目标变量预测贡献的期望变化。
*   **关键技术细节**：
    *   **模型架构假设**：方法适用于**可加、可分解贡献的自回归模型**（如NAVAR）。在这类模型中，对目标变量 `j` 的预测可以分解为每个源变量 `i` 的独立贡献之和：`X_j(t) = Σ f_ij(X_i(t-1...t-K))`。
    *   **形式化定义**：将边 `i → j` 的**函数值因果影响**定义为：
        `g_ij(x) = E[贡献_ij(t) | 源变量_i在t-1时刻的值为x]`。这是一个关于 `x` 的函数，描述了当源变量状态为 `x` 时，它对目标变量的平均贡献。
    *   **估计方法 - 个体条件期望**：为了从训练好的模型中估计出 `g_ij(x)`，作者使用了**个体条件期望**。这是一种类似“干预”的模型探测方法：
        1.  遍历标准的 `x` 值网格。
        2.  对于每个 `x`，将模型中所有关于源变量 `i` 的输入（所有滞后项）都强制设为 `x`。
        3.  计算在这种“干预”下，模型对目标变量 `j` 的预测变化 `Δ`。
        4.  对所有时间窗口的 `Δ` 求平均，得到 `hat{g}_ij(x)` 的估计值。
    *   **状态条件版本**：为了捕捉不同状态下的效应差异，可以引入状态变量 `r`（如目标变量的历史值），并计算条件化的个体条件期望曲线`hat{g}_ij(x | r)`，从而揭示阈值、不对称性等状态依赖效应。

#### 6. 论文的主要结论与发现

*   **核心发现**：在非线性时间序列因果模型中，**标量因果分数是严重的信息瓶颈**。它可以反映因果关系的存在性或强度，但无法刻画其功能形态。
*   **合成实验**：证明了具有几乎相同标量分数的有向边，其背后的真实因果函数可以是线性、阈值、饱和或符号翻转等完全不同的形式。
*   **真实案例分析**：在民主发展面板数据中，标量分数表现出相似的弱效应值，而个体条件期望曲线则揭示了强烈的状态依赖性：
    *   **阈值效应**：某些变量（如言论自由）对选举质量的积极影响只有在一个最低制度基线被超越后才显现。
    *   **不对称效应**：制度改善与恶化带来的影响速度或幅度不同。
    *   **饱和效应**：在高度民主化阶段，进一步改善的边际回报会递减。
*   **总结**：非线性因果模型的“黑箱”问题，部分源于输出表示方式而非模型本身。通过保留模型已学会的**函数值结构**，可以恢复可解释的、状态特定的因果机制。
