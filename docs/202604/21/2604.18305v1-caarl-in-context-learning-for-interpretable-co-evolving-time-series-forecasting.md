## 摘要
本文利用大语言模型（LLM）方法研究具有复杂依赖关系和非平稳动力学特征的协同演化时间序列预测。我们提出了一种名为上下文感知 ARLLM（CAARL）的新型建模方法，该方法提供了一个可解释的框架，用于解码影响协同演化序列变化的上下文动力学。CAARL 将时间序列分解为自回归片段，构建时间依赖图，并将该图序列化为叙述性文本，以便由 LLM 处理。这种设计产生了一种类似于思维链（chain-of-thought）的推理路径，其中中间步骤捕捉上下文动力学，并以透明的方式引导预测。通过将预测与显式推理轨迹相结合，CAARL 在保持准确性的同时增强了可解释性。在真实世界数据集上的实验验证了其有效性，使 CAARL 成为现有最先进预测方法的一种具有竞争力的、可解释的替代方案。

## Abstract
In this paper we investigate forecasting coevolving time series that feature intricate dependencies and nonstationary dynamics by using an LLM Large Language Models approach We propose a novel modeling approach named ContextAware ARLLM CAARL that provides an interpretable framework to decode the contextual dynamics influencing changes in coevolving series CAARL decomposes time series into autoregressive segments constructs a temporal dependency graph and serializes this graph into a narrative to allow processing by LLM This design yields a chainofthoughtlike reasoning path where intermediate steps capture contextual dynamics and guide forecasts in a transparent manner By linking prediction to explicit reasoning traces CAARL enhances interpretability while maintaining accuracy Experiments on realworld datasets validate its effectiveness positioning CAARL as a competitive and interpretable alternative to stateoftheart forecasting methods

### 1. 核心问题与研究动机
*   **核心问题**：现实世界的时间序列（如金融、交通、医疗）通常是**协同演化**的，即多个序列之间存在复杂的相互依赖关系，且具有**非平稳动态**特征。
*   **研究动机**：
    *   传统的自回归（AR）模型虽具可解释性，但难以处理非平稳和复杂的跨序列依赖。
    *   深度学习模型（如 GNN、Transformer）虽然预测精度高，但往往是“黑盒”，缺乏透明度，难以让用户信任。
    *   **目标**：开发一个既能捕捉复杂协同演化关系，又能提供可靠、可理解解释的预测模型。

### 2. 方法论
CAARL 框架分为三个核心步骤：
*   **步骤 1：上下文场景创建（In-context scenario creation）**
    *   **模型识别**：将时间序列划分为不同区间，在每个区间内用简单的自回归模型（AR）拟合。通过聚类算法（Clust）识别出一组代表性的自回归参数集（模式）。
    *   **依赖建模**：构建一个**时序依赖图（Temporal Graph）**。节点代表不同的自回归模型，边代表时间序列在不同区间之间切换模型的转换关系。这捕捉了序列间的同步行为和潜在的演化逻辑。
*   **步骤 2：图序列化（Graph Serialization）**
    *   **核心思想**：将抽象的时序依赖图转化为自然语言描述（叙述性文本）。
    *   **技术细节**：描述特定序列在过去几个时间步的模型切换历史，以及它与其他序列的同步行为（例如：“序列 A 和 B 在 T1 到 T2 期间都使用了模型 0”）。
*   **步骤 3：预测任务（Prediction Task）**
    *   **LLM 推理**：将序列化文本输入大语言模型（如 GPT-4）。
    *   **分层预测**：首先让 LLM 判断序列在下一阶段是否会切换模型（Yes/No）；如果切换，预测将切换到哪个模型；最后利用预测出的 AR 模型参数计算具体的数值预测。
