## 摘要
语言模型可用于识别文本中金融实体之间的关系。然而，尽管存在结构化输出机制，基于提示（prompting）的流水线仍会产生自回归解码成本，并将图构建与下游优化解耦。我们提出了“关系探测”（Relational Probing），它将标准的语言模型头替换为关系头，直接从语言模型的隐藏状态中归纳出关系图，并与用于股票趋势预测的下游任务模型进行联合训练。该方法既学习了语义表示，又保留了所归纳关系图的严格结构。它使语言模型的输出能够超越文本，允许将其重塑为适用于下游模型的特定任务格式。为了增强可复现性，我们提供了小语言模型（SLM）的操作性定义：即在指定的批大小和序列长度设置下，可以在单个 24GB GPU 上进行端到端微调的模型。实验使用 Qwen3 骨干网络（0.6B/1.7B/4B）作为上游 SLM，并与共现基线进行对比。关系探测在具有竞争力的推理成本下实现了持续的性能提升。

## Abstract
Language models can be used to identify relationships between financial entities in text. However, while structured output mechanisms exist, prompting-based pipelines still incur autoregressive decoding costs and decouple graph construction from downstream optimization. We propose \emph{Relational Probing}, which replaces the standard language-model head with a relation head that induces a relational graph directly from language-model hidden states and is trained jointly with the downstream task model for stock-trend prediction. This approach both learns semantic representations and preserves the strict structure of the induced relational graph. It enables language-model outputs to go beyond text, allowing them to be reshaped into task-specific formats for downstream models. To enhance reproducibility, we provide an operational definition of small language models (SLMs): models that can be fine-tuned end-to-end on a single 24GB GPU under specified batch-size and sequence-length settings. Experiments use Qwen3 backbones (0.6B/1.7B/4B) as upstream SLMs and compare against a co-occurrence baseline. Relational Probing yields consistent performance improvements at competitive inference cost.

股票市场预测不仅依赖于个股的动量，还受到公司间复杂关系（如供应链、行业竞争、持股关系）的影响。虽然大型语言模型（LLM）擅长从新闻中提取实体关系，但现有方法存在两个主要缺陷：

*   **任务解耦：** 图构建过程与下游的预测任务是分离的，语言模型优化的目标（如文本流畅度）并不一定对最终的金融预测任务最优。
**研究动机：** 提出一种直接从语言模型隐藏状态诱导关系图的方法，实现图构建与下游预测任务的端到端联合优化。

### 2. 方法论：核心思想与关键技术
论文提出了 **Relational Probing（关系探测）** 框架，其核心流程如下：
1.  **新闻编码（News Encoding）：** 使用小语言模型（SLM）作为骨干网络，提取新闻文本的上下文隐藏状态 $H$。
2.  **关系诱导（Relation Induction）：** 
    *   引入可学习的股票代码嵌入（Ticker Embeddings）。
    *   通过**交叉注意力机制**，让股票嵌入与新闻的隐藏状态进行交互，生成受新闻内容调节的股票表示。
    *   使用**双线性映射（Bilinear Map）**计算股票对之间的交互得分，生成单篇新闻的关系矩阵。
3.  **每日图构建：** 将同一天内所有新闻生成的矩阵进行聚合与归一化，通过阈值处理生成稀疏的邻接矩阵。
4.  **下游预测（GAT）：** 将诱导出的动态图输入图注意力网络（GAT），结合历史交易数据（由 LSTM 编码的节点特征），预测次日股价趋势（上涨、下跌、平盘）。
5.  **联合训练：** 预测误差通过 GAT 反向传播至关系头（Relation Head）乃至语言模型，使关系提取直接服务于预测目标。

### 6. 主要结论与发现
*   **性能提升：** 关系探测生成的图在 Accuracy、Macro F1、MCC 和 AUC 等指标上均显著优于传统的“共现”基线。

### 7. 优点
*   **端到端对齐：** 解决了图构建与下游任务脱节的问题，使模型能学到对预测“有用”的关系，而非仅仅是事实上的关系。
*   **推理高效：** 摆脱了 LLM 慢速的文本生成环节，直接在向量空间操作，适合高频金融场景。
*   **可复现性强：** 明确了 SLM 的硬件边界和实验配置，对算力受限的研究者非常友好。
