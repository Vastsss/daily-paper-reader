## 摘要
我们提出了 Brainstacks，这是一种用于大语言模型持续多领域微调的模块化架构。该架构将领域专业知识封装为冻结的适配器堆栈，并在推理时以加性方式组合在共享的冻结基座上。该系统由五个相互关联的组件组成：(1) MoE-LoRA，在 QLoRA 4 位量化和 rsLoRA 缩放的基础上，对所有七个 Transformer 投影层采用 Shazeer 风格的带噪声 top-2 路由；(2) 内循环，通过冻结已训练的堆栈并添加新堆栈来执行残差提升（residual boosting）；(3) 外循环，训练具有课程顺序依赖关系的序列化领域特定堆栈；(4) 零空间投影，通过随机 SVD 将新堆栈约束在与先前方向正交的子空间中，从而在孤立状态下实现零遗忘；(5) 基于结果的 Sigmoid 元路由器，在经验发现的领域组合目标上进行训练，通过选择性加权堆栈实现跨领域组合。两项边界实验：(6) 在随机初始化模型上进行 PSN 预训练；(7) 逐领域强化学习（DPO/GRPO），验证了与 SFT 后对齐的兼容性。在 TinyLlama-1.1B（4 个领域，9 个堆栈）和 Gemma 3 12B IT（5 个领域，10 个堆栈）上的验证表明，MoE-LoRA 的收敛速度比参数匹配的单一 LoRA 快 2.5 倍，残差提升突破了单堆栈的性能上限，且路由系统恢复了因无门控堆栈累积而受损的生成质量。核心发现：基于结果的路由器发现，领域堆栈编码的是可迁移的认知原语（指令遵循清晰度、数值推理、过程逻辑、思维链结构），而非单纯的领域特定知识；在 97% 的案例中，医疗提示词被路由到了“聊天+数学”堆栈，尽管这些堆栈中完全不包含医疗数据。

## Abstract
We present Brainstacks, a modular architecture for continual multi-domain fine-tuning of large language models that packages domain expertise as frozen adapter stacks composing additively on a shared frozen base at inference. Five interlocking components: (1) MoE-LoRA with Shazeer-style noisy top-2 routing across all seven transformer projections under QLoRA 4-bit quantization with rsLoRA scaling; (2) an inner loop performing residual boosting by freezing trained stacks and adding new ones; (3) an outer loop training sequential domain-specific stacks with curriculum-ordered dependencies; (4) null-space projection via randomized SVD constraining new stacks to subspaces orthogonal to prior directions, achieving zero forgetting in isolation; (5) an outcome-based sigmoid meta-router trained on empirically discovered domain-combination targets that selectively weights stacks, enabling cross-domain composition. Two boundary experiments: (6) PSN pretraining on a randomly initialized model; (7) per-domain RL (DPO/GRPO) validating compatibility with post-SFT alignment. Validated on TinyLlama-1.1B (4 domains, 9 stacks) and Gemma 3 12B IT (5 domains, 10 stacks), MoE-LoRA achieves 2.5x faster convergence than parameter-matched single LoRA, residual boosting breaks through the single-stack ceiling, and the routed system recovers generation quality destroyed by ungated stack accumulation. The central finding: the outcome-based router discovers that domain stacks encode transferable cognitive primitives (instruction-following clarity, numerical reasoning, procedural logic, chain-of-thought structure) rather than domain-specific knowledge, with medical prompts routing to chat+math stacks in 97% of cases despite zero medical data in those stacks.

### Brainstacks：通过冻结的 MoE-LoRA 堆栈实现大语言模型持续学习的跨领域认知能力

#### 1. 核心问题与研究动机
当前大语言模型（LLM）的微调方法主要是“单体式”的，即在混合数据集上进行单次训练。这导致了三个核心问题：
*   **灾难性遗忘**：添加新领域知识时，必须重新训练或面临旧知识丢失。
*   **缺乏模块化**：部署后无法独立更新或删除特定领域的模型能力。
*   **推理冗余**：无论输入为何，模型总是激活所有参数，无法根据需求选择性调用专业知识。
论文旨在通过一种模块化的架构，使模型能够像“堆积木”一样持续吸收新领域能力，同时保证零遗忘和跨领域的能力组合。

#### 2. 方法论：核心思想与关键技术
Brainstacks 构建在五个互锁的组件之上：
*   **MoE-LoRA 构建块**：在 QLoRA 4-bit 量化基础上，对 Transformer 的所有 7 个投影矩阵应用混合专家 LoRA。采用 Shazeer 风格的带噪声 Top-2 路由和 rsLoRA（秩稳定缩放）。
*   **内循环（残差提升）**：在同一领域内，训练并冻结一个堆栈后，添加新堆栈来学习前一堆栈未捕获的残差误差，从而突破单堆栈的性能瓶颈。
*   **外循环（持续领域训练）**：按课程顺序（聊天 -> 代码 -> 数学 -> 医疗 -> 推理）序列化训练领域堆栈。
*   **零空间投影（Null-Space Projection）**：利用随机 SVD 计算先前堆栈激活空间的投影矩阵，强制新堆栈在正交的子空间中更新。这在数学上保证了不同领域间的干扰被物理隔离，实现孤立评估下的“零遗忘”。
*   **基于结果的元路由器（Outcome-based Meta-Router）**：一个轻量级网络，通过观察不同堆栈组合对损失函数的实际降低效果（而非仅根据领域标签）来学习路由。它使用 Sigmoid 激活，允许同时激活多个领域堆栈进行能力组合。

#### 3. 实验设计
*   **实验对象**：TinyLlama-1.1B（4 领域，9 堆栈）和 Gemma 3 12B IT（5 领域，10 堆栈）。
*   **数据集**：Alpaca (Chat), Python Code Instructions (Code), GSM8K (Math), MedQA/PubMedQA (Medical), OpenThoughts/LogiQA (Reasoning)。
*   **Benchmark**：HellaSwag, ARC, TruthfulQA, MMLU, GSM8K, MedQA, MedMCQA 等 8 个零样本基准测试。
*   **对比方法**：参数匹配的单一 LoRA（rank=64）、未经过微调的基座模型、以及无门控（所有堆栈同时开启）的累积模型。

#### 4. 资源与算力
*   **TinyLlama 实验**：总训练时长约 485 分钟（约 8 小时）。
*   **Gemma 3 12B 实验**：在 Google Colab G4 实例上运行，配备 96GB 显存（推测为 A100 或类似级别）。
*   **推理优化**：提出了“叠加 LLM”（Superposition LLM）原则，通过将冻结堆栈离线存储在磁盘并按需加载，实现了在恒定 GPU 显存下支持无限领域堆栈。

#### 5. 实验数量与充分性
论文进行了多维度的实验：
*   **消融实验**：对比了有无零空间投影的效果，验证了正交性对减少干扰的作用。
*   **收敛性测试**：对比了 MoE-LoRA 与单 LoRA 的收敛速度。
*   **边界实验**：包括在随机初始化模型上的 PSN 预训练实验，以及逐领域的强化学习（DPO/GRPO）兼容性测试。
*   **充分性评价**：实验覆盖了从 1B 到 12B 不同规模的模型，且包含了从 SFT 到 RL 的全流程验证，实验设计较为客观、严谨。

#### 6. 主要结论与发现
*   **认知原语发现**：这是论文最核心的发现。领域堆栈学习的是“认知原语”（如指令遵循、数值推理、逻辑结构），而非单纯的领域知识。例如，医疗提示词在 97% 的情况下会路由到“聊天+数学”堆栈，因为这些堆栈提供了处理医疗问题所需的逻辑和格式能力。
*   **性能提升**：MoE-LoRA 的收敛速度比单 LoRA 快 2.5 倍；残差提升显著降低了验证损失。
*   **路由的必要性**：无门控的堆栈累积会导致严重的性能下降（量级干扰），而元路由器能有效恢复生成质量。
*   **零遗忘**：通过物理冻结和数学正交约束，实现了架构层面的知识保护。

#### 7. 优点与亮点
*   **架构创新**：将 MoE、LoRA、残差提升和零空间投影有机结合，提供了一套完整的持续学习方案。
*   **理论深度**：重新定义了微调的本质——注入可组合的认知能力，而非简单的知识检索。
*   **实用性强**：磁盘卸载推理机制解决了多适配器部署时的显存瓶颈。
*   **零遗忘保证**：通过硬性的几何约束而非软性的正则化来处理遗忘问题。

#### 8. 不足与局限
*   **推理延迟**：由于需要逐个计算冻结堆栈的增量并可能涉及 CPU-GPU 之间的数据交换，推理延迟会随堆栈数量增加。
*   **容量上限**：虽然目前 12B 模型能容纳数十个领域，但随着领域增加，隐藏层的维度空间最终会耗尽。
*   **预训练依赖**：PSN 实验表明该架构依赖于已有相干几何结构的预训练基座，无法从零开始高效构建模块化能力。
*   **路由训练敏感**：元路由器的效果高度依赖于训练数据的质量和多样性，容易受到数据污染的影响。
