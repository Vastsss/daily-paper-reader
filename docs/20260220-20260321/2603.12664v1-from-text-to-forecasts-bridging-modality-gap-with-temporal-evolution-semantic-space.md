## 摘要
将文本信息融入时间序列预测有望解决事件驱动的非平稳性；然而，一个根本性的模态差距阻碍了有效的融合：文本描述隐式且定性地表达时间影响，而预测模型则依赖于显式且定量的信号。通过受控的半合成实验，我们表明现有方法过度关注冗余标记，且难以可靠地将文本语义转化为可用的数值线索。为了弥合这一差距，我们提出了 TESS，它引入了一个时间演化语义空间（Temporal Evolution Semantic Space）作为模态之间的中间瓶颈。该空间由可解释的、具有数值基础的时间原语（均值偏移、波动性、形状和滞后）组成，这些原语由大语言模型（LLM）通过结构化提示从文本中提取，并通过置信度感知门控进行过滤。在四个真实世界数据集上的实验表明，与最先进的单模态和多模态基准模型相比，预测误差降低了高达 29%。代码将在论文接收后发布。

## Abstract
Incorporating textual information into time-series forecasting holds promise for addressing event-driven non-stationarity; however, a fundamental modality gap hinders effective fusion: textual descriptions express temporal impacts implicitly and qualitatively, whereas forecasting models rely on explicit and quantitative signals. Through controlled semi-synthetic experiments, we show that existing methods over-attend to redundant tokens and struggle to reliably translate textual semantics into usable numerical cues. To bridge this gap, we propose TESS, which introduces a Temporal Evolution Semantic Space as an intermediate bottleneck between modalities. This space consists of interpretable, numerically grounded temporal primitives (mean shift, volatility, shape, and lag) extracted from text by an LLM via structured prompting and filtered through confidence-aware gating. Experiments on four real-world datasets demonstrate up to a 29 percent reduction in forecasting error compared to state-of-the-art unimodal and multimodal baselines. The code will be released after acceptance.

*   **核心问题**：如何有效利用外部文本信息（如新闻、社交媒体）来增强时间序列预测，特别是应对由突发事件引起的**非平稳性（Non-stationarity）**。
*   **研究背景**：现有的多模态预测方法通常直接融合文本嵌入和数值特征，但存在严重的**模态差距（Modality Gap）**。文本是定性、非结构化且包含大量冗余的，而预测模型需要定量、精确的数值信号。
*   **关键发现**：作者通过受控的半合成实验证明，现有模型容易被文本中的冗余信息干扰（注意力错位），且即使去除冗余，模型也难以直接从自然语言中解码出稳定的预测增益（表示失配）。

#### 2. 方法论：TESS 框架
TESS（Temporal Evolution Semantic Space）提出了一个两阶段框架，引入“中间瓶颈”来转化信息：
*   **核心思想**：不直接融合原始文本，而是将文本蒸馏为一组具有数值基础的**时间演化原语（Temporal Evolution Primitives）**。
*   **关键技术细节**：
    1.  **定义语义空间**：预定义了四类统计学关键原语：
        *   **均值偏移（Mean Shift）**：预测未来数值水平的变化。
        *   **波动性（Volatility）**：预测数据波动的剧烈程度。
        *   **形状（Shape）**：预测趋势形态（如上升、下降、峰值、谷值）。
        *   **滞后与衰减（Lag and Decay）**：预测影响发生的时间点和持续性。
    2.  **文本到原语的投影**：利用冻结的大语言模型（LLM，如 GPT-4/5）通过结构化提示（Prompting）对文本进行分类，提取上述原语。
    3.  **置信度感知门控（Confidence-aware Gating）**：由于 LLM 提取可能出错，引入一个可学习的门控网络，根据 LLM 输出的概率分布（Top-1 与 Top-2 的差值）估计置信度，对不可靠的原语进行软加权抑制。
    4.  **原语条件预测**：将加权后的语义向量作为“前缀标记（Prefix Tokens）”输入到基于 PatchTST 的 Transformer 预测器中，引导数值预测。
